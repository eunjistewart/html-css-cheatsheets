Get specific

add a class or ID label to an element's HTML tag, then refer to that label in the CSS.

ex) Use classes and ids

ID - lets us specify a unique label so that we can opply distinct styles to just one thing. In CSS, use # in front of the ID names.

CSS - lets us label several elements as belonging to the same group. You can use the same class throughout your HTML in order to apply the same style to many elements, then target that class in CSS using . (dot) in front of the class names.


Display Basics

All HTML elements have some default styling. Likewise, all HTML elements have a default way of being displayed on a page: either as a block element or an inline element.

Block vs. Inline

Block- always on a new line; takes up the full width; stack on top of each other when new ones are added; set margins, padding, widths, or heights; examples are h1 ~ h6 tags, div, p, ul, etc

inline- sits within the content surrounds; only take up as necessary; render in one line; cannot have set margins/padding ; examples are link a tags, span, img, etc


Changing the Display

ex) natually inline img => block
natually block ul => all in one line inline

Display: none; element won't display at all.

Display: inline; display in one horizontal line.

Display: block; stacked & take up the entire width of page.

Display: inline-block; sit next to each other (inline), but can have margin, padding, width, and height properties(block).
